ISTG-33 Family Control Name SYSTEM USE NOTIFICATION
ISTG-33 Supplemental Guidance System use notifications can be implemented using messages or warning banners displayed before individuals log in to information systems. System use notifications are used only for access via logon interfaces with human users and are not required when such human interfaces do not exist. Organizations consider system use notification messages/banners displayed in multiple languages based on specific organizational needs and the demographics of information system users. Organizations also consult with the Departmental Legal Services Branch for legal review and approval of warning banner content
