ISTG-33 Family Control Name PREVIOUS LOGON (ACCESS) NOTIFICATION
ITSG-33 Control Reference
ISTG-33 Supplemental Guidance This control enhancement permits organizations to specify additional information to be provided to users upon logon including, for example, the location of last logon. User location is defined as that information which can be determined by information systems, for example, IP addresses from which network logons occurred, device identifiers, or notifications of local logons
