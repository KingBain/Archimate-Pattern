ITSG-33 Control Reference
ITSG-33 Parent Class Technical
ISTG-33 Family Control Name AUDIT REDUCTION AND REPORT GENERATION
Artifact Location
ISTG-33 Supplemental Guidance Events of interest can be identified by the content of specific audit record fields including, for example, identities of individuals, event types, event locations, event times, event dates, system resources involved, IP addresses involved, or information objects accessed. Organizations may define audit event criteria to any degree of granularity required, for example, locations selectable by general networking location (e.g., by network or sub-network) or selectable by specific information system component. Related controls: AU-2, AU-12.
