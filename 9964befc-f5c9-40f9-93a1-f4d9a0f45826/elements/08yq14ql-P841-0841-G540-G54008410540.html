ISTG-33 Family Control Name MONITORING PHYSICAL ACCESS
ITSG-33 Control Reference
ITSG-33 Parent Class Operational
ISTG-33 Supplemental Guidance This control enhancement focuses on recording surveillance video for purposes of subsequent review, if circumstances so warrant (e.g., a break-in detected by other means). It does not require monitoring surveillance video although organizations may choose to do so. Note that there may be legal considerations when performing and retaining video surveillance, especially if such surveillance is in a public location.
