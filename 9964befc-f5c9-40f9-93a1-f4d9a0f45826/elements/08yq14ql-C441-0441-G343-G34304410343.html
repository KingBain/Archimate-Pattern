ITSG-33 Control Reference TBS Operational Security Standard on Physical Security [Reference 6]. TBS Operational Security Standard - Management of Information Technology Security [Reference 7].
ISTG-33 Family Control Name INFORMATION SYSTEM BACKUP
ISTG-33 Supplemental Guidance System-level information includes, for example, system-state information, operating system and application software, and licenses. User-level information includes any information other than system-level information. Mechanisms employed by organizations to protect the integrity of information system backups include, for example, digital signatures and cryptographic hashes. Protection of system backup information while in transit is beyond the scope of this control. Information system backups reflect the requirements in contingency plans as well as other organizational requirements for backing up information. Related controls: CP-2, CP-6, MP-4, MP-5, SC-13
